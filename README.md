# 🐋 Black Whale Academy — نظام تفقد الحضور

تطبيق خفيف لتفقد حضور الأطفال بنادي الحوت الأسود، مبني بـ Next.js وقاعدة بيانات Postgres (Neon).

## المكونات

- **/register** — رابط تسجيل يُرسل للأهل، يعبّوا اسم الطفل + بيانات تواصل + صورة، وبيتولّد له كود QR خاص فيه.
- **/admin** — إدارة المجموعات (اسم المجموعة + المشرف)، وتوزيع الأطفال على المجموعات، وطباعة بطاقة QR لكل طفل.
- **/attendance** — شاشة المشرف اليومية: مسح QR من كاميرا الموبايل لتسجيل الحضور، مع إمكانية تعديل يدوي.
- **/history** — استعراض سجل أي يوم سابق حسب المجموعة.

## خطوات التشغيل

### ١. قاعدة البيانات (Neon)

1. افتح مشروع Postgres جديد على [neon.tech](https://neon.tech).
2. افتح **SQL Editor** وشغّل محتوى ملف `schema.sql` الموجود بهذا المستودع (مرة وحدة فقط).
3. انسخ الـ **Connection String** (رابط الاتصال).

### ٢. النشر على Vercel

1. افتح [vercel.com](https://vercel.com) وسجل دخول بحساب GitHub.
2. اعمل **Import Project** واختر هذا المستودع.
3. بقسم **Environment Variables** ضيف:
   ```
   DATABASE_URL = <رابط الاتصال يلي نسخته من Neon>
   ```
4. اضغط **Deploy**. بعد ما يخلص، رح تاخد رابط live جاهز.

### ٣. الاستخدام

- شارك `<رابطك>/register` مع الأهل.
- من `<رابطك>/admin` ضيف المجموعات، ووزّع الأطفال، واطبع بطاقات QR.
- كل مشرف بيفتح `<رابطك>/attendance` من موبايله ويختار مجموعته.

## التطوير محلياً

```bash
npm install
cp .env.example .env.local   # وحط رابط قاعدة بياناتك
npm run dev
```
